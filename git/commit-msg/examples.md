# commit-msg examples

## stage モード — ステージ差分から生成

**コマンド**
```
/commit-msg stage
```

**出力例**
```
feat: ログイン画面にパスワード強度チェック機能を追加

- 入力中にリアルタイムで強度インジケーターを表示
- 8文字未満・記号なしの場合は送信ボタンを無効化
- バリデーションロジックをPasswordValidatorクラスに分離
```

---

## stage モード — context（チケット番号）付き

**コマンド**
```
/commit-msg stage ABC-456
```

**出力例**
```
fix: カート削除時のnullチェック漏れを修正 (ABC-456)

- CartItemが空の場合にNullPointerExceptionが発生していた問題を修正
- 削除処理前にnullガードを追加
```

---

## latest モード — 直近コミットをまとめて1コミットに集約

**コマンド**
```
/commit-msg latest
```

**出力例**
```
refactor: 認証ミドルウェアをクラスに切り出し

- 各ルートに散在していた認証ロジックを AuthMiddleware に集約
- トークン検証・ロール確認をメソッドとして分離
- 既存のエンドポイント動作に影響なし
```

---

## commit モード — 特定コミットIDから生成

**コマンド**
```
/commit-msg commit a1b2c3d
```

**出力例**
```
docs: API仕様書にエラーコード一覧を追記

- 400・401・403・500 の説明と対処方法を追加
- サンプルレスポンスを各エラーコードに付記
```
